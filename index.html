Shopping Cart App (Local Run)

Full-stack Node.js (v20) + Express v5 app with MongoDB/Mongoose and Passport.js session auth. Includes REST endpoints, a GraphQL API (Yoga at /graphql), role-based access (admin/user), session cart, and checkout flow.

  • Stack: Node.js, Express v5, MongoDB/Mongoose, Passport.js, Handlebars, GraphQL Yoga
  • Status: Local run (repo includes setup + seed)

GitHub Repo

How to run locally 
git clone https://github.com/dvelasquez124/shopping-cart-app
cd shopping-cart-app
npm install
cp .env.example .env   # set MONGODB_URI, SESSION_SECRET
npm run seed           # optional: adds admin + sample products
npm run dev            # or npm start
# UI:        http://localhost:3000
# GraphQL:   http://localhost:3000/graphql
