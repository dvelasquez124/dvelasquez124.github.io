🌍 WanderMom Travel Blog

Live Site: https://playpackrepeat.netlify.app

GitHub Repo: wandermom_blog

This colorful, family-centered travel blog showcases responsive design, component reuse, and personal branding. Features include:

  • ✅ Fully responsive layout with custom mobile styles
  • 🎨 Dynamic blog content and toggles using Vue.js
  • 🎥 Multimedia integration with video and audio
  • 📋 Expandable content and styled tables
  • 📝 Accessible and styled contact form with JavaScript validation

This project highlights front-end development with HTML, CSS, and JavaScript (Vue), along with Git/GitHub deployment and Netlify hosting.

📚 Featured Project: Student Support Portal

Student Support Portal

A beautifully styled, responsive web app that helps parents and educators discover engaging, age-appropriate, and autism-friendly educational resources.

  • 🔗 Live App: student-support-portal.netlify.app
  • 🌐 Umbrella GitHub Repo: the-student-support-portal
  • 💻 Frontend: student-support-frontend
  • 🛠️ Backend: student-support-backend

Note: The app may take a few seconds to load because the backend is hosted on Render's free tier, which goes to sleep when idle.

🛠 Tech Stack

  • Frontend: HTML, TailwindCSS, Vanilla JavaScript, Vue.js
  • Backend: Django, Django REST Framework
  • Styling & UI: Custom CSS, responsive media queries, animations
  • Deployment: Netlify (frontend), Render (backend)
  • Version Control: Git, GitHub
