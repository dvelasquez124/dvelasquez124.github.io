📚 Featured Project

Student Support Portal

A beautifully styled, responsive web app that helps parents and educators discover engaging, age-appropriate, and autism-friendly educational resources.

  • 🔗 Live App: student-support-portal.netlify.app
  • 🌐 Umbrella GitHub Repo: the-student-support-portal
  • 💻 Frontend: student-support-frontend
  • 🛠️ Backend: student-support-backend

Note: The app may take a few seconds to load because the backend is hosted on Render's free tier, which goes to sleep when idle.

🛠 Tech Stack

  • Frontend: HTML, TailwindCSS, Vanilla JavaScript
  • Backend: Django, Django REST Framework
  • Deployment: Netlify (frontend), Render (backend)
